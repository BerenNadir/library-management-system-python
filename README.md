# Library Management System (Python)

A command-line library management system developed in Python.  
The application allows users to manage books and users in a simple library environment.

## Features

- Add new books
- Register library users
- Borrow and return books
- View available books
- File-based data storage

## Technologies

- Python
- File handling

## Project Structure

The system uses functions and file handling to store and manage library data.  
It was designed to practice programming fundamentals and data management.

## How to Run

1. Clone the repository
2. Open the project in any Python IDE or editor
3. Run the main Python file
